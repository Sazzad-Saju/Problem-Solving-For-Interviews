## Problem Description ##

Problem 16: Given an array and an integer k, rotate the array such that the last k elements are moved to the front

<br>
<h3> Example: </h3>

```
Input:
array = [
    [1, 2, 3, 4, 5],
]
K = 2
Output: [3,4,5,1,2]

Input:
array = [
    [1, 2, 3, 4, 5],
]
K = 3
Output: [4,5,1,2,3]

```
