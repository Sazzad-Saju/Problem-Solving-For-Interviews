## Problem Description ##

Problem 07: String Recursive Permutation

Find the array of all possible combinations of a given string
<br>
<h3> Example: </h3>

```
Input: abc
Output: 
[                                                                               > 
  [ 'a', 'b', 'c' ],
  [ 'a', 'c', 'b' ],
  [ 'b', 'a', 'c' ],
  [ 'b', 'c', 'a' ],
  [ 'c', 'a', 'b' ],
  [ 'c', 'b', 'a' ] 
]
```
